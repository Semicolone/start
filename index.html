Team Project Description

팀명 Semicolone
프로젝트명 AI 질문을 ‘기억되는 인사이트’로 바꿔주는 개인 지식 관리 플랫폼
프로젝트 소개 AI 기반 질문 기록 및 인사이트 내재화 플랫폼을 만들고자 한다.

최근 많은 사람들이 ChatGPT와 같은 생성형 AI를 통해 다양한 질문을 하고 정보를 얻고 있지만, 대부분의 대화 내용은 일회성으로 소비되고 이후에는 다시 활용되지 않는 경우가 많다.

본 서비스는 사용자가 AI에게 질문했던 내용과 답변을 기록하고, 이를 AI가 자동으로 정리하고 연결하며, 시간이 지나 다시 노출하여 장기 기억으로 내재화하도록 돕는 개인 지식 관리 시스템이다.

단순히 AI 대화를 저장하는 것이 아니라 사용자가 얻은 인사이트를 정리하고 필요할 때 다시 활용할 수 있도록 돕는 것을 목표로 한다.
해결하려는 문제 최근 생성형 AI 사용이 증가하면서 사용자들은 다양한 질문을 통해 많은 정보를 얻고 있다.

그러나 대부분의 AI 대화는 일회성으로 끝나고 시간이 지나면 어떤 질문을 했는지, 어떤 답변을 받았는지 기억하기 어렵다.

또한 AI 대화 기록은 길고 복잡한 경우가 많아 필요한 정보를 다시 찾기 어렵고, 중요한 인사이트가 묻히는 문제가 발생한다.

이로 인해 사용자는 많은 질문을 통해 정보를 얻지만 실제로 자신의 지식으로 축적되거나 활용되는 경우는 제한적이라는 문제가 발생한다.
주요 기능 사용자는 ChatGPT와 같은 AI에게 질문했던 내용을 복사하여 서비스에 입력한다.

사용자가 질문과 답변을 저장하면 시스템은 해당 내용을 분석하여 핵심 인사이트 요약과 키워드를 자동으로 생성한다.

생성된 키워드를 기반으로 인사이트는 자동으로 분류된다.

새로운 인사이트가 저장되면 시스템은 기존에 저장된 질문 및 인사이트와의 의미적 유사성을 분석하여 관련된 인사이트를 추천한다.

시간이 지나면 시스템은 일정한 간격으로 이전에 저장한 인사이트를 다시 보여주어 사용자가 내용을 기억하도록 돕는다.

사용자는 키워드나 검색 기능을 통해 과거에 저장한 질문과 인사이트를 다시 확인할 수 있다.
기술 스택 프론트: React, React Native
백엔드: Node.js (Express.js)
AI: Groq Cloud API (LPU 가속 기반 Llama 3.3 모델)
데이터 저장: PostgreSQL (관계형 데이터베이스)
기타: JSON Mode 구조화 응답, Vector Embedding (Next Phase 예정)
